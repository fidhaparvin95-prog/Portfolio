# ConnectFour

## Project Overview
ConnectFour is a web-based variant of the classic Connect Four game, developed to improve student engagement.  
The project allows multiple users to play, track scores, and interact via a web interface.

This project is built with **PHP**, **JavaScript**, **HTML**, **CSS**, and uses a **MySQL database** to store user data, game progress, and scores.

---

## Features
- User registration and login system  
- Real-time game play tracking  
- Scoreboard and player statistics  
- Profile management  
- Interactive and responsive user interface  

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Libraries & Frameworks:** Bootstrap, jQuery
